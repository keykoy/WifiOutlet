# WifiOutlet
## 目的 ##
适逢硬件开源及模块化的潮流, 加上树莓派这样mini而强大的操作系统. 出于个人爱好, 打造一套围绕**[Springer](https://github.com/keykoy/Springer)**(运行在树莓派上的管理系统)的智能家居平台.
## 说明 ##

**WifiOutlet** 是一款基于ESP8266模块的无线插座.
